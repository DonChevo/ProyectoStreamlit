# README
El proceso de aprendizaje en las clases de Python fue sumamente enriquecedor, permitiéndome consolidar una base sólida desde la sintaxis elemental hasta la estructuración avanzada de software. Esta asimilación conceptual se tradujo directamente en el desarrollo de este Proyecto 1, donde logré aplicar y materializar conceptos abstractos —como la inmutabilidad en programación funcional, la manipulación eficiente de matrices con NumPy y el encapsulamiento de datos mediante la Programación Orientada a Objetos— dentro de un entorno web interactivo real. Durante la construcción de la aplicación, las principales dificultades radicaron en la gestión de la persistencia de datos en memoria debido al ciclo de vida reactivo de Streamlit, lo que exigió un control riguroso del estado de la sesión (st.session_state) y de los flujos de ejecución (st.rerun()) para sincronizar las operaciones CRUD. Sin embargo, considero que el mayor beneficio de esta experiencia de aprendizaje fue la capacidad de transformar código de consola en una solución de software tangible, escalable y visual, destacando el valor de la modularidad y preparándome con competencias técnicas clave para afrontar desafíos reales en el desarrollo de soluciones analíticas.
