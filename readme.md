# Proyecto 1: Fundamentos de Python en Streamlit

## Descripción del Proyecto
Esta es una aplicación interactiva diseñada para demostrar el dominio de los fundamentos de Python. A través de una interfaz visual limpia y amigable, la app unifica conceptos clave del lenguaje —desde variables y estructuras de datos hasta programación funcional y orientada a objetos (POO)— en herramientas prácticas y funcionales que el usuario puede probar en tiempo real.

## Lo que aprendí y los desafíos del proyecto
El proyecto me ha permitido transformar conceptos abstractos como programación funcional, manejo de datos y Programación Orientada a Objetos en una aplicación web real e interactiva. 
Un desafío técnico fue entender el ciclo de vida de Streamlit, como por ejemplo usar st.session_state y st.rerun() para que los datos no se borraran al interactuar con la pantalla.

## Estructura del Repositorio
* `app.py`: Archivo principal de la aplicación Streamlit.
* `libreria_funciones.py`: Módulo local con las funciones utilitarias del proyecto.
* `libreria_clases_proyecto1.py`: Módulo local que contiene la lógica de Programación Orientada a Objetos (POO).
* `requirements.txt`: Lista de dependencias externas requeridas para el despliegue.
